# Aprendendo git

##comandos básicos do git

```sh
git init # para iniciar um repositório
git remote add origin link_do_repositório_do_github # pegar link no github
git add . # adiciona todos os arquivos para serem monitorados
git commit -m "com uma mensagem legal entre  aspas"
git push origin main # envia dos dados para o github
git pull origin main# recebe arquivos modificados ou excluidos.
#também informa se o repo remoto está atualiizado com o local e vice-versa
git status
```
